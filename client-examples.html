üéØ Method 1: Data Attributes (Recommended)

Best for: Most websites, easy configuration, clean implementation

Basic Integration:

<!-- Add this script tag to your website --> <script src="https://your-domain.com/chatwidget.js" data-client-id="bawma_co_uk" data-client-name="Bawma Clinic" data-subtitle="Expert Medical Care" data-description="Get instant answers about our services and treatments" data-placeholder="What can we help you with today?" data-primary-color="#2563eb" data-secondary-color="#1d4ed8"> </script>

Full Configuration Options:

Attribute Description Example
data-client-id Unique identifier for client (required) bawma_co_uk
data-client-name Business name displayed in header Bawma Clinic
data-subtitle Subtitle below business name Expert Medical Care
data-description Welcome message description Get instant answers about our services
data-placeholder Input field placeholder text What can we help you with today?
data-primary-color Main brand color (hex) #2563eb
data-secondary-color Secondary color for gradients #1d4ed8
data-profile-icon URL to profile image (optional) https://example.com/logo.png
data-show-refresh-button Show refresh button (true/false) true
data-webhook-url Custom webhook URL (optional) https://custom-n8n.com/webhook/...

üìã Integration Instructions

For Clients:

  1. Choose your configuration method (data attributes recommended)
  2. Customize the colors and text to match your brand
  3. Add the script tag to your website (before closing </body> tag)
  4. Test on staging before going live
  5. Deploy to production

Where to place the code:

<!DOCTYPE html> <html> <head> <title>Your Website</title> </head> <body> <!-- Your website content --> <!-- Add chat widget before closing body tag --> <script src="https://your-domain.com/chatwidget.js" data-client-id="your_client_id" data-client-name="Your Business" data-primary-color="#your-color"> </script> </body> </html>
Note: The widget will automatically appear in the bottom-right corner of the website. It's mobile-responsive and won't interfere with existing website functionality.
